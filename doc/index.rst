Overview
--------

**PyMODM** is a generic ODM on top of `PyMongo`_, the MongoDB Python
driver. This documentation attempts to explain everything you need to know to
use **PyMODM**.

.. _PyMongo: https://pypi.python.org/pypi/pymongo
